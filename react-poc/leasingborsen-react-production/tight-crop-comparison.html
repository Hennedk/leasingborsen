Key Improvements

  • Tighter Cropping: Reduced padding from 15% to 5% for main processed image
  • Minimal Standard Padding: Reduced from 5% to 2% for grid/detail views
  • Higher Crop Limit: Increased max crop from 80% to 90% allowing more aggressive cropping
  • Smaller Minimum Padding: Reduced from 50px to 20px absolute minimum
  • 16:9 Aspect Ratio: Changed grid to 800x450 and detail to 1920x1080 (Full HD)

Result

The new configuration produces images much closer to the LeaseLoco style with:

  • ✅ Minimal whitespace around vehicles
  • ✅ Tighter, more professional cropping
  • ✅ Better use of available image space
  • ✅ Standard 16:9 aspect ratio for modern displays
  • ✅ Full HD detail images (1920x1080)
